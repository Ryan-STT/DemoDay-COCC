CÓDIGO

O COCC (Centro de Oportunidade Cultural à Criança) é um projeto desenvolvido em Java com o propósito de criar um site para crianças de 5 a 12 anos com conteúdo cultural de músicas, livros, desenhos, jogos etc. O projeto é estruturado em HTML5, CSS3, Java e JavaScript com DataBase MySQL

SITE COCC

O projeto foi criado com a intenção de levar conteúdo de forma virtual nesse período de pandemia para às crianças se manterem dentro de suas casas, se divertindo enquanto aprendem. Porém, mesmo após esse período pandemico continuaremos levando entretenimento cultural e educacional para as crianças conhecerem e reviverem suas raízes culturais

APOIADORES & INFORMAÇÕES

Em base nas nossas pesquisas, podemos ver quantos conteúdos encontramos na internet, mas muito deles não são apropriados ou agregam algo para o futuro de nossas crianças, pensando nisso, fomos em busca de profissionais da área infantil para apresentar nosso projeto e receber feedbacks, para assim seguir um caminho correto e direto para o desenvolvimento infantil com ainda mais qualidade. Além disso, manter os pais confiantes do nosso trabalho e propósito, pois todo o conteúdo disponivel para seus filhos irá passar por uma fase de avaliação
