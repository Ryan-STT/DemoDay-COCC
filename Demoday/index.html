CÓDIGO
O COCC (Centro de Oportunidade Cultural a Criança) é um projeto desenvolvido em Java com o proposito de criar um site para crianças de 5 a 12 anos com conteúdo cultural de músicas, livros, desenhos, jogos e etc. O projeto é estruturado em HTML5, CSS3, JavaScript com DataBase MySQL.
